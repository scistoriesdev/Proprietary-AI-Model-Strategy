Low-Rank Adaptation (LoRA)

üîê

Proprietary

You own the .safetensors file containing your trained style.

üîÑ

Portable

Works with any SDXL/Flux base model. Switch bases without retraining.

üß©

Stack multiple LoRAs (Style + Subject) for granular control.

Style LoRA (The "Studio Look")

Captures aesthetic: "Flat Vector Icon," "3D Cross-Section"

Dataset:30-50 images
Rank:16-32 (low)
Focus:Lighting, line weight, shading

Subject LoRA (The "Morphology")

Captures structure: "Mitochondria," "Ribosome"

Dataset:15-20 images
Rank:64-128 (high)
Focus:Cristae folds, membrane details

Training Workflow

1. Dataset Preparation

  • ‚Ä¢ 15-50 high-res images (min 1024√ó1024)
  • ‚Ä¢ Remove blurry/inconsistent samples
  • ‚Ä¢ For icons: solid background (#FFFFFF)

2. Captioning

ohwx_style, mitochondria, cross-section, cristae, medical illustration

3. Training Config

  • ‚Ä¢ Use OneTrainer or Kohya_ss
  • ‚Ä¢ Enable gradient checkpointing
  • ‚Ä¢ Save checkpoints every 5 epochs

4. Validation (XY Plot)

  • ‚Ä¢ Test multiple epochs + weights
  • ‚Ä¢ Best is usually 60-80% through training
  • ‚Ä¢ Avoid final checkpoint (often overfit)

The 80/20 Rule for Training Art

80% Consistency

Same style, color palette, perspective, line weight, background treatment

20% Variation

Different subjects, compositions, scales, orientations

Technical Requirements by Style

  • ‚Ä¢ GPU: RTX 3060 12GB+ (RTX 4070+ recommended)
  • ‚Ä¢ RAM: 16GB minimum, 32GB+ recommended
  • ‚Ä¢ Python: 3.10.11 from python.org

Linux Setup (Local Training)

Prerequisites
  • ‚Ä¢ OS: Ubuntu 22.04/24.04 LTS
  • ‚Ä¢ Kernel: 5.15+ (6.5+ for latest NVIDIA)
  • ‚Ä¢ GPU: RTX 3060 12GB+

# NVIDIA Driver Installation

sudo apt update

cd OneTrainer
pip install -r requirements.txt
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121
python scripts/train_ui.py
